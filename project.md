# Markoff Photos — Описание проекта

> Последнее обновление: апрель 2026

---

## Что это за проект

Личный сайт-портфолио фотографа Аркадия Маркова. Одностраничный лендинг, цель которого — привлекать частных клиентов и бизнес, показывать стиль и уровень работ, формировать доверие и конвертировать посетителей в клиентов через Telegram. Сайт заменяет дорогостоящую рекламу: работает как круглосуточный менеджер по продажам.

---

## Для кого

**Частные клиенты:**
- Пары, планирующие лав стори или свадьбу
- Семьи с детьми (заказывают семейные и детские съёмки)
- Люди, ищущие портрет для соцсетей / личного бренда
- Возраст: 25–45 лет, преимущественно женщины — инициаторы съёмки

**Бизнес-клиенты:**
- Малый бизнес для предметной и коммерческой съёмки
- Кафе, магазины, бренды, цветочные студии

## Дизайн-система

### Платформа
Пока: чистый HTML + CSS + JS (один файл)  
Рекомендация для продакшена: **Tilda** (план Business, ~1 250 ₽/мес) — либо оставить кастомный код на хостинге

### Цветовая палитра
| Роль | HEX |
|---|---|
| Основной фон | `#FAF8F5` (молочный белый) |
| Вторичный фон | `#F0EDE8` (тёплый светло-серый) |
| Основной текст | `#2C1A0E` (тёмно-коричневый) |
| Вторичный текст | `#8C7B72` (тёплый серый) |
| Акцент | `#B07D5E` (терракот) |
| Тёмный футер | `#1A110A` |

### Шрифты
- **Cormorant Garamond** — заголовки (тонкие, воздушные, с засечками)
- **Inter** — основной текст (чистый sans-serif)

### Настроение
Тёплый минимализм · Воздушный · Живой · Настоящий · Премиум без пафоса

---

## Технические требования (для продакшена)
- Заменить все фото-плейсхолдеры на WebP ≤ 400 КБ
- Hero-фото прелоадить `<link rel="preload">`
- Lazy loading для всех фото кроме hero
- PageSpeed Score ≥ 80 (мобайл), ≥ 90 (десктоп)
- SSL-сертификат (HTTPS)
- Мобильная версия протестирована на iOS Safari + Android Chrome
